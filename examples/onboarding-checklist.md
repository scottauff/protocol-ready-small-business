# Protocol-Ready SMB Onboarding Checklist (Example)

## Identity
☑ Generate business Nostr keypair  
☑ Back up private key in secure offline medium  
☑ Publish npub on website & collateral  
☑ Enable Nostr DMs for support  
☑ Verify signing and release process  

## Treasury
☑ Set up self-custodial Bitcoin wallet  
☑ Set up Lightning wallet for settlement  
☑ Fund wallet with test amount  
☑ Confirm inbound/outbound invoice flow  
☑ Confirm static Lightning address works  

## V4V Consulting
☑ Establish per-minute advisory model  
☑ Create micro-invoice workflow  
☑ Train staff on V4V UX expectations  

## Open Catalog
☑ Publish catalog JSON endpoint  
☑ Validate schema for machine readability  
☑ Mirror catalog listing as a Nostr event  
☑ Verify discovery visibility (relay search)  

## Fulfillment Transparency
☑ Define SLA artifact format (timestamp, deliverable metrics)  
☑ Capture first job deliverable as signed artifact  
☑ Publish anonymized example for transparency  

## Dispute Flow
☑ Define escalation steps  
☑ Define refund logic and settlement path  
☑ Capture first dispute artifact (anonymized)  

## Participation
☑ Invite onboarding UX tester  
☑ Reward tester via zap  
☑ Record feedback and checklist improvements
